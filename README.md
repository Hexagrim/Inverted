# Inverted
A game about inverting the world to solve puzzles. super short and made just as a concept!

***

## Controls
Movement: A and D
Jump: Space
Invert: E
Reset Level: R
Quit To Menu: Esc

***

## Levels

Since this is a concept and i got too tired to make levels, only 11 super easy and fun levels are there.
